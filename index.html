About Student Finance Tracker

Purpose

This Student Finance Tracker helps students manage their expenses, track spending patterns, and stay within budget. Built with vanilla HTML, CSS, and JavaScript, it demonstrates modern web development practices including accessibility, responsive design, offline functionality, and clean modular architecture.

Features

  • - Complete expense tracking with categories
  • - Advanced regex search and validation
  • - Responsive mobile-first design
  • - Full keyboard accessibility
  • - Data import/export (JSON & CSV)
  • - Multi-currency support
  • - Spending caps and budget tracking
  • - Real-time statistics dashboard
  • - Dark/Light theme toggle
  • - Offline-first functionality
  • - Progressive Web App (PWA)

Keyboard Navigation

Tab - Navigate through interactive elements

Shift + Tab - Navigate backwards

Enter/Space - Activate buttons and links

Escape - Close dialogs or cancel editing

Arrow keys - Navigate within lists and tables

Alt + T - Toggle dark/light theme

Alt + 1/2/3 - Navigate to Dashboard/Records/Add-Edit

Ctrl + Enter - Submit form (when in Add/Edit section)

Technology Stack

Built with modern web standards:

  • Semantic HTML5 with ARIA accessibility
  • CSS3 with custom properties and responsive design
  • Vanilla ES6+ JavaScript with modules
  • Service Worker for offline functionality
  • Progressive Web App features
